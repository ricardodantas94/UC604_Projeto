# Portefólio - Ricardo Dantas

Website de portefólio pessoal desenvolvido com HTML e CSS.

Este projeto apresenta o meu percurso profissional, competências e alguns trabalhos na área da publicidade, produção gráfica e desenvolvimento web.

## Funcionalidades

* Apresentação pessoal
* Secção de competências e experiência
* Apresentação de projetos
* Página de contacto com formulário

## Tecnologias

* HTML5
* CSS3

## Estrutura

* `index.html` – página principal
* `contactos.html` – página de contacto
* `principal.css` – estilos principais
* `contactos.css` – estilos da página de contacto

## Como usar

Abrir o ficheiro `index.html` no navegador.

## Contacto

Disponível através da página de contacto incluída no projeto.
